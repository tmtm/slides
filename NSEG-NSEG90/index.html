reveal-ck

https://github.com/jedcn/reveal-ck

明らかにckは..

reveal.jsのプレゼンテーションの作成に役立つコマンドラインツールです。

これにより、reveal.jsプレゼンテーションの構築の詳細を処理することで、プレゼンテーションの内容に焦点を当てることができます。

あなたはマークダウンで1つのファイルにプレゼンテーションを書き、reveal-ckは他のものを生成します。

reveal.jsスライドをmarkdownで簡単に作れるツール

インストール

※ 要Ruby

% gem install reveal-ck

空スライドを作る

% touch slides.md
% reveal-ck generate
% reveal-ck serve

ブラウザから localhost:10000 にアクセス

あとはブラウザ見ながら slides.md を編集するだけ
